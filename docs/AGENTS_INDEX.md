# AGENTS Docs Index

Root `AGENTS.md` is the canonical source. These files are extracted slices meant to reduce context pressure for local-model work.

- `AGENTS_PROJECT_CONTEXT.md` - project overview, board, colors, piece types
- `AGENTS_REMOVAL_RULES.md` - line scoring rules
- `AGENTS_ATTACK_RULES.md` - capture and attack constraints
- `AGENTS_KING_RULES.md` - king behavior, spawn, movement, pawn rule
- `AGENTS_ARCHITECTURE.md` - engine version, folder map, asset layout
- `AGENTS_CONSTANTS.md` - shared board constants and color mapping
- `AGENTS_RENDERING.md` - sprites, theme system, visual scope
- `AGENTS_MAIN_MENU.md` - menu layout and settings behavior
- `AGENTS_RESIZE_LAYOUT.md` - viewport scaling and layout rules
- `AGENTS_HUD_PUZZLE_BOARD.md` - HUD, puzzle board, pause flow, messages
- `AGENTS_DIALOGS.md` - dialog presentation rules
- `AGENTS_INPUT_HANDLING.md` - click flow and input ownership
- `AGENTS_MOVE_VALIDATION.md` - legal move rules and capture separation
- `AGENTS_LINE_DETECTION.md` - line detector behavior
- `AGENTS_HIGHLIGHTING.md` - move and selection overlays
- `AGENTS_GAME_STATE_OWNERSHIP.md` - responsibility boundaries between scripts
- `AGENTS_SCORING_RULES.md` - score structure and bonuses
- `AGENTS_PIECE_DISTRIBUTION.md` - random generation and inventory limits
- `AGENTS_DEVELOPMENT_RULES.md` - edit/refactor constraints
- `AGENTS_TESTING.md` - syntax checks and manual checklist
- `AGENTS_CURRENT_WORKING_DEFINITION.md` - concise end-state summary
